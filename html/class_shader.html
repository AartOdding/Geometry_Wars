Detailed Description

Shader class, represents an OpenGL shader program.

This class represents a shader program in OpenGL. The OpenGL state represented by this class is mostly immutable, and should be modified through the ShaderState class. The layout of the shader can be specified, which will only effect ShaderState's that are produced with this shader.
