Detailed Description

template<typename KEY_TYPE, typename VALUE_TYPE>
class MapView< KEY_TYPE, VALUE_TYPE >

MapView: Class that is only used internally, to create read only view of a map.

Allows the owner to let external users access a std::map through operator[] without letting the external users change the map itself. Used by ShaderState to grant access to the uniform and attribute variables.
